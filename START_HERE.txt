🆘 HAVING ISSUES?
================================================================================

Setup fails?
  → Check Node.js is installed: node --version
  → Check Android Studio is installed
  → Read CAPACITOR_BUILD_GUIDE.md

Build fails?
  → Run: npm run build
  → Run: npm run android:build again
  → Read ANDROID_CONFIG.md troubleshooting

APK won't install?
  → Enable "Unknown sources" on phone
  → Check APK file size (should be 18-22 MB)
  → Try: adb install android/app/build/outputs/apk/debug/app-debug.apk

Notifications don't work?
  → Click "Test Notification" in app
  → Check phone notification settings
  → Verify app has notification permission

Need help?
  → Check relevant .md file
  → Search for your error
  → Read troubleshooting sections
